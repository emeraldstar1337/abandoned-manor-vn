# Заброшенная Усадьба — Интерактивный Хоррор

Интерактивная визуальная новелла (текстовый квест) с атмосферным оформлением, звуковым сопровождением и скрытыми концовками.

## Особенности проекта
* **Процедурный звук**: Фоновые шумы, сердцебиение, звуки скрипов и шагов генерируются динамически через **Web Audio API** прямо в браузере.
* **Иллюстрации**: Атмосферная графика сгенерирована нейросетью Imagen.
* **Нелинейный сюжет**: Несколько выборов, инвентарь и 3 разные концовки (включая секретную с ножом).
* **Стильный интерфейс**: Glassmorphism, анимация пламени, пульсация темноты и адаптивная верстка.

## Локальный запуск
Так как браузеры блокируют некоторые локальные ресурсы по соображениям безопасности (CORS), рекомендуется запускать проект через локальный веб-сервер:

### Вариант 1: Через Python (уже установлен)
Запустите в папке проекта:
```bash
python3 -m http.server 8000
```
И откройте в браузере: `http://localhost:8000`

### Вариант 2: Через Node.js
```bash
npx serve .
```

## Публикация на GitHub Pages

Чтобы залить игру на GitHub и дать поиграть другим:

1. **Создайте новый пустой репозиторий** на [GitHub](https://github.com/new) с именем `abandoned-manor-vn`.
2. **Свяжите локальный репозиторий с удаленным** и отправьте код:
   ```bash
   git remote add origin https://github.com/ВАШ_НИКНЕЙМ/abandoned-manor-vn.git
   git branch -M main
   git push -u origin main
   ```
3. **Включите хостинг GitHub Pages**:
   * Перейдите в настройки репозитория на GitHub: **Settings** -> **Pages**.
   * В разделе **Build and deployment** выберите в качестве источника (Source): **Deploy from a branch**.
   * Укажите ветку **main** (папка `/ (root)`) и нажмите **Save**.
   * Спустя 1-2 минуты игра будет доступна по адресу: `https://ВАШ_НИКНЕЙМ.github.io/abandoned-manor-vn/`
